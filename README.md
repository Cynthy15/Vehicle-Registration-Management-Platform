# Vehicle Registration Management Platform

## Overview
The Vehicle Registration Management Platform is a web-based application designed to simplify and digitize the process of registering, managing, and tracking vehicles. It allows users and administrators to efficiently handle vehicle records with a modern and user-friendly interface.

---

## Features

- Register new vehicles
- Search and view registered vehicles
- Update vehicle details
- Delete vehicle records
- Notifications (using react-hot-toast)
- Modern UI with Tailwind CSS
- Dashboard

---

## Technologies Used

### Frontend
- React (Vite)
- Tailwind CSS
- React Router DOM
- React Query
- Lucide React (icons)
- React Hot Toast (notifications)

### Backend 
API from backend
